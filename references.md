# 参考文献概要

### [1] CSE-CIC-IDS2018 数据集主页（UNB）
该页面给出 CSE‑CIC‑IDS2018 的官方背景与构建理念，核心是通过“行为画像（profiles）”系统化生成入侵流量，使数据能覆盖多协议、多拓扑与多业务场景。页面说明数据集包含正常流量与多类入侵行为，并提供原始流量与日志，同时给出由 CICFlowMeter‑V3 抽取的 80 项流量特征，便于机器学习建模与对比评估。对本项目而言，该来源用于支撑数据集合法性、特征来源与实验可复现路径，并为“降维+分类”任务提供权威数据背景。

### [2] CSE-CIC-IDS2018（AWS Open Data Registry）
该条目由 AWS Open Data Registry 维护，说明数据集由 CSE 与 CIC 协作建设，采用结构化场景设计并包含多类攻击。条目强调数据集提供网络流量与日志，并明确 80 项特征由 CICFlowMeter‑V3 抽取，适用于入侵检测机器学习研究。条目还给出公开托管信息与引用方式，强化数据可获取性与来源可信度。该文献可作为报告中“数据来源与可获取性”的官方依据，并为数据规模与攻击覆盖的描述提供支撑。

### [3] Sharafaldin et al., 2018, ICISSP
该论文指出早期 IDS 数据集在攻击覆盖、流量多样性与现实性上存在不足，提出构建面向现实网络的入侵检测数据集，并详细描述网络拓扑、攻击场景与特征提取流程。文中对多类流量特征与多种机器学习算法进行评估，展示不同攻击类别在特征选择与检测性能上的差异。该研究为本项目的“特征筛选依据”“降维维度设置”与“分类器对比逻辑”提供理论支撑，并可用于说明数据集设计的合理性与代表性。

### [4] CICFlowMeter 应用说明（UNB）
CICFlowMeter 介绍页面说明该工具可生成双向流，并在前向与后向方向分别计算 80 余项统计特征，覆盖时长、包数、字节数、速率与包长分布等关键指标。该来源解释了 CSV 特征文件的生成机制，也说明了为何无需处理原始 PCAP 即可完成模型训练与评估。对报告而言，它支持“特征工程来源”的可追溯性，以及“选择流量统计特征进行降维与分类”的技术合理性。

### [5] Jolliffe, 2002, Principal Component Analysis (2nd ed.)
该书系统阐述 PCA 的数学基础与统计解释，强调以最大方差方向构造低维子空间，从而在信息保留与维度压缩间取得平衡。书中还讨论了 PCA 的算法实现、方差解释率与模型诊断思路，为降维后信息保留率的计算与解释提供依据。该文献可用于报告中对 PCA 的理论描述、主成分贡献率的定义以及“降维维度选择”理由的学术支撑。

### [6] Fisher, 1936, The Use of Multiple Measurements in Taxonomic Problems
该论文是线性判别分析（LDA）的经典源头之一，提出在多指标测量下构建线性判别函数以最大化类别间可分性。作者通过对多变量测量的统计推导展示了判别方向与类别区分度之间的关系，为监督降维与分类分析奠定基础。该文献为本项目采用 LDA 进行降维与类别分离度评估提供权威依据，也可用于解释 LDA 相较 PCA 在类别信息保留上的优势。

### [7] van der Maaten & Hinton, 2008, Visualizing Data using t-SNE
论文提出 t‑SNE 作为高维数据可视化方法，改进了传统 SNE 的优化难度与“拥挤”问题，通过保持局部邻域结构实现更清晰的簇结构呈现。作者展示了 t‑SNE 在多种数据集上的可视化优势，并强调其在多尺度结构上保持类内紧密与类间分离的能力。该文献为本项目的二维可视化分析与“降维后类别可分性”的图形解释提供理论支撑。

### [8] Cortes & Vapnik, 1995, Support-vector Networks
该文献提出支持向量机的核心思想：通过核映射将样本映射到高维特征空间，并构造最大间隔的线性分离超平面，以提升泛化能力。论文还讨论了不可线性分离情形下的软间隔与误差容忍机制，为实践中的核函数选择与惩罚参数设定提供理论基础。该来源可用于报告中 SVM 分类器的理论动机、超参数意义与泛化优势说明。

### [9] Breiman, 2001, Random Forests
论文提出随机森林作为决策树集成模型，强调单树强度与树间相关性共同决定泛化误差，并通过随机特征选择降低相关性以提升整体性能。文中还给出内建误差估计与变量重要性评估方法，说明随机森林在噪声与复杂非线性关系下的鲁棒性。该文献为本项目采用随机森林及其特征重要性解释提供原始依据。

### [10] Hosmer, Lemeshow & Sturdivant, 2013, Applied Logistic Regression (3rd ed.)
该书系统介绍逻辑回归的模型假设、参数估计与诊断方法，强调用协变量解释二元结果概率的优势，并扩展到交互项、模型拟合优度与应用型评估。书中内容强调模型可解释性与统计推断价值，适合作为基线分类器的理论支撑。该文献可用于报告中逻辑回归的建模流程、阈值设定与误报/漏报分析的解释基础。
